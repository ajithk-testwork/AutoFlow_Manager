AutoFlow Manager

A professional MERN stack vehicle service and monthly payment collection management system for car wash and vehicle maintenance businesses.

✨ Features
Monthly payment collection management
Dynamic pending amount calculation
Missed payment date tracking
WhatsApp payment reminder & thank you messages
Customer & vehicle management
Admin & seller dashboards
Product/service approval workflow
Global payment history
QR code support
Cloudinary image upload
Responsive modern dashboard UI
🔄 Workflow
Seller Registration
        ↓
Admin Verification
        ↓
Seller Dashboard Access
        ↓

        ↓

        ↓
